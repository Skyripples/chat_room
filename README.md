# 匿名聊天室

網頁版匿名聊天室

https://skyripples.github.io/chat_room/

## 主要功能

- 進入聊天室前需要設定暱稱，暱稱不能和目前在線使用者重複。
- 所有人預設會進入公開聊天室。
- 可以建立聊天室；未設定密碼就是公開聊天室，設定密碼後就是私人聊天室。
- 私人聊天室需要輸入正確密碼才能進入。
- 每個 IP 最多只能建立一個聊天室。
- 建立聊天室時可以選擇限制人數，未設定時預設最多 100 人。
- 聊天室建立者可以刪除自己建立的聊天室。
- 右側會顯示目前聊天室成員，自己會固定在最上方。
- 可以點選聊天室成員傳送私人訊息。
- 每個 IP 每秒最多只能發送一則訊息。
- 每則訊息最多 100 個字。
- 公開聊天室訊息會保留 7 天。
- 自訂聊天室沒有人時會自動刪除。
